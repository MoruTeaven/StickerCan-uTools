# 表情罐头 - uTools表情包管理插件

一个功能强大的uTools表情包管理插件，支持云存储、标签管理和快速搜索。

## 功能特性

- 🎯 **智能搜索**：通过标签快速搜索表情包
- ☁️ **云存储**：支持S3对象存储，节省本地空间
- 🏷️ **标签管理**：灵活的标签系统，便于分类管理
- 📋 **一键复制**：快速复制表情包到剪贴板
- 🔄 **数据同步**：通过uTools云端同步设置和表情包数据
- 📱 **响应式设计**：适配不同屏幕尺寸

## 安装方法

1. 下载插件文件到本地
2. 在uTools中打开"插件管理"
3. 点击"安装插件"
4. 选择插件文件夹
5. 完成安装

## 使用方法

### 添加表情包

1. 点击右下角的 ➕ 按钮
2. 选择添加方式：
   - **URL方式**：直接输入图片URL和标签
   - **上传方式**：上传本地图片到云存储

### 搜索表情包

1. 在顶部搜索框输入关键词
2. 支持模糊搜索标签
3. 实时显示搜索结果

### 管理表情包

1. 点击表情包卡片查看详情
2. 支持以下操作：
   - 📋 复制到剪贴板
   - 🏷️ 编辑标签
   - 🗑️ 删除表情包
